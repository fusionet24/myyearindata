Real-World Hook Patterns

🚫
PII Blocking
Pre-hook scans DAX/SQL for customer names, emails, or personal identifiers and blocks execution
🔒
Read-Only Enforcement
Pre-hook rejects any query containing INSERT, UPDATE, DELETE, or DROP statements
👥
Row-Level Security
Pre-hook injects WHERE clauses based on the authenticated user's security context
📋
Audit Logging
Post-hook records every tool call — parameters, results, who triggered it, and when
⚠
Output Sanitisation
Post-hook masks credit card numbers, national IDs, and sensitive data in results
🛠
Cost Controls
Pre-hook enforces query complexity limits and token budgets per session
