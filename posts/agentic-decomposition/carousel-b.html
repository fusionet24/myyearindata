The #1 Mistake

Common Anti-Pattern

One Agent Per Tool

An agent that only calls one API isn't autonomous.
It's just a wrapper with extra latency.

Tool access is a capability, not a reason to exist.

If it doesn't make decisions, it's not an agent.
