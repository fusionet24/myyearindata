Indirect Prompt Injection

Malicious instructions hidden in data your agent processes:

Code comments • GitHub issues • Log files • Web pages

It's not your prompt. It's theirs.

The agent reads it. The agent follows it.

DNS Data Exfiltration

"Safe" commands like dig and nslookup can encode your environment secrets in DNS queries.

Sent to attacker-controlled servers.

Your allow-list is the attack vector.

Well done, you trusted the wrong commands.

Clickfix for Robots

Social engineering works on LLMs too.

Unicode Invisibility

Self-Modification

Sets tools.approved: true

#6

Agent Hopper Virus

Self-replicating exploit that spreads through Git:

Infects repo ➔ Modifies other repos ➔ Pushes to GitHub ➔ Next dev pulls ➔ Repeats

Your agent just infected your other projects.

Old-school virus, new-school delivery mechanism.
